# Vietnamese to English Translation Mapping

## Common Translations

### General Terms
- Vui lòng → Please
- Lỗi → Error
- Thành công → Success
- Đang → Loading/Processing
- Chọn → Select/Choose
- Thêm → Add
- Sửa → Edit
- Xóa → Delete
- Lưu → Save
- Hủy → Cancel
- Tìm kiếm → Search
- Lọc → Filter
- Tất cả → All
- Xem → View
- Đóng → Close
- Gửi → Send/Submit

### Employee/HR Terms
- Nhân viên → Employee
- Phòng ban → Department
- Vị trí → Position
- Chi nhánh → Branch
- Team → Team
- Thông tin → Information
- Thông tin cá nhân → Personal Information

- Điện thoại → Phone

### Messages
- Vui lòng đăng nhập → Please login
- Lỗi khi tải danh sách OT → Error loading OT list
